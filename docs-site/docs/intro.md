# metax-llm 文档

本仓库用于“微调开源大模型进行问答评测”，并提供符合评测系统契约的推理服务（HTTP）。

核心目标：

- 准确率：RougeL-F1（jieba 分词），常见参考阈值 ≥ 0.35
- 速度：尽量提高吞吐（tokens/s），以在限定时间内完成更多题目

评测契约（必须保持）：

- `GET /`：健康检查（需要快速返回）
- `POST /predict`：请求 `{"prompt":"..."}`，响应 `{"response":"..."}`
- 端口：`8000`

重要约束：

- Build 阶段允许联网（下载依赖/权重）
- Run 阶段断网（请求路径内不要做任何联网）
